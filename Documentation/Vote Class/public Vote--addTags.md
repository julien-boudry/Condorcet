## public Vote::addTags

### Description    

```php
public $Vote -> addTags ( mixed tags ) : bool
```

Add tag(s) on this Vote.    


##### **tags:** *mixed*   
Tag(s) are non-numeric alphanumeric string. They can be added by string separated by commas or an array.    



### Return value:   

In case of success, return TRUE
