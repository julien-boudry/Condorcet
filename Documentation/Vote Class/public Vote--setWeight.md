## public Vote::setWeight

### Description    

```php
public Vote->setWeight ( int newWeight ): int
```

Set a vote weight. The vote weight capacity must be active at the election level for producing effect on the result.
    

##### **newWeight:** *int*   
The new vote weight.    


### Return value:   

*(int)* New weight.
