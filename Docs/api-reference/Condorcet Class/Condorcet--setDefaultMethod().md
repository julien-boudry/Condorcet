## Description    

```php
public static Condorcet::setDefaultMethod ( string $method ): bool
```

Put a new static method by default for the news Condorcet objects.

## Parameter

### **method:** *`string`*   
A valid method name or class.    


## Return value   

*(`bool`)* In case of success, return TRUE
